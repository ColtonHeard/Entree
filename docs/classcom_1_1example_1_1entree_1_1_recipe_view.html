Detailed Description

Represents a double list view where the top section contains a horizontally scrolling list of IngredientChips and the bottom section contains a vertically scrolling list of RecipeCards.

Member Function Documentation

◆ addChip()

void com.example.entree.RecipeView.addChip ( String  label)

Adds an ingredient chip with the given label. The first two ingredients are added to the top LinearLayout, the second two to the bottom. All subsequent chips are added as follows: if there are an even number of chips currently, it is added to the bottom, otherwise it is added to the top LinearLayout. This results in a bottom heavy look to the chips.

Parameters
labelThe label to set the IngredientChip's text to.
